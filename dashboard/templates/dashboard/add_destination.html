Comment ça fonctionne ?

Lorsque vous ajoutez une nouvelle destination, vous devrez ensuite lancer manuellement une première collecte de données pour cette destination.

Exemples de destinations
  • Paris,France
  • Londres,Royaume-Uni
  • Barcelone,Espagne
  • New York,États-Unis
  • Tokyo,Japon
Après l'ajout

Une fois la destination ajoutée, vous serez redirigé vers sa page détaillée où vous pourrez lancer le scraping en cliquant sur le bouton "Mettre à jour les données".

La collecte des données peut prendre plusieurs minutes. Vous pouvez quitter la page et revenir plus tard pour consulter les résultats.
